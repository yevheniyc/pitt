# Project 2: Map Visualization

## Overview

Geographic data visualization project using Python geospatial libraries to create interactive maps and analyze spatial patterns.

## Files

- `Chuba-Yevheniy-M12-Map-Visualization-Notebook.ipynb` - Main map visualization assignment (2,993 lines)

## Key Concepts

- **Geospatial Data Processing**: Working with geographic datasets
- **Interactive Mapping**: Creating dynamic, user-interactive maps
- **Spatial Analysis**: Understanding geographic patterns and relationships
- **Cartographic Principles**: Effective map design and visualization

## Technical Stack

- **GeoPandas**: Geospatial data manipulation
- **Folium**: Interactive map generation
- **Shapely**: Geometric operations
- **Matplotlib**: Static map visualization
- **Geodatasets**: Access to geospatial datasets

## Learning Outcomes

- Master geospatial data visualization techniques
- Create professional-quality interactive maps
- Apply spatial analysis methods
- Understand geographic data formats and projections
